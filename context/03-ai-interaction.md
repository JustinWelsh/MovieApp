# AI Interaction

## Working on TODOs and Features

When working through `current-todos.md` or `current-feature.md`:

1. **State which item is in progress** at the start — one item at a time.
2. **Tackle one todo/feature at a time.** Do not move on until the current one is done and committed.
3. **After completing an item**, provide a brief summary of what was done, then ask for permission to commit.
4. **Follow Commit Conventions** from `02-coding-standards.md` (Conventional Commits: `type: short description`).

**If permission to commit is given:**

1. Run `npm run build` to confirm no issues — skip if all changed files are `.md` files.
2. Commit the work (no `Co-Authored-By` lines).
3. Update `current-todos.md` or `current-feature.md` to mark the item complete.
